HTML Tags and Elements

HTML uses tags to define elements on a web page.

Tags vs. Elements

A tag is a code snippet that defines an HTML element, while an element is the actual content that appears on the web page.

Basic HTML Tags

  • <html> - defines the root of an HTML document
  • <head> - contains metadata about the document
  • <title> - defines the title of the document
  • <body> - contains the visible content of the document
  • <h1> - defines a heading
  • <p> - defines a paragraph
  • <a> - defines a hyperlink
  • <img> - defines an image

These are just a few of the many HTML tags that you can use to create a web page.
